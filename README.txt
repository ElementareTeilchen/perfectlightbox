This is the extension "perfectlightbox" to be used within Typo3.
Title: Perfect Lightbox2
Description: Click-enlarge images with PROTACULOUS/LIGHTBOX or MOOTOOLS/SLIMBOX in Typo3. Single images, imagesets, presentation mode and slideshow supported. Options can be set for each content-element. No XCLASS, just typoscript. Valid html. Flexible configuration.
